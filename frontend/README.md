# Scheduler Platform (Cal.com Clone)

## Features
- Event creation
- Availability management
- Dynamic slot generation
- Booking system
- Prevents double booking
- Public booking page

## Tech Stack
- Frontend: React + Tailwind
- Backend: Node.js + Express
- Database: MySQL

## Setup Instructions

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm run dev
